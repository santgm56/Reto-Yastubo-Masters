# Customer Portal Assets

Coloca aqui los assets visuales especificos del portal customer.

Archivos sugeridos para la siguiente fase:

- `user-avatar-default.jpg`
  Placeholder para el avatar del usuario cuando no exista foto real.

- `user-avatar-christian.jpg`
  Avatar de referencia para pruebas visuales del dashboard.

- `hero-beneficiary-placeholder.jpg`
  Imagen opcional si luego reemplazamos iniciales por foto en beneficiarios.

- `action-report-fallecimiento.jpg`
  Imagen opcional para la card de reportar fallecimiento, si se decide acercarla mas al mockup.

Regla de uso:

- Usa dimensiones cuadradas para avatares, idealmente `400x400`.
- Usa JPG o PNG optimizado y nombres estables.
- Mantener peso bajo para no penalizar la carga del portal.
